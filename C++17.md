📦 C++17 — Complete Feature List


---

🧠 1. Core Language Features


---

✅ Structured Bindings

auto [x, y] = std::pair<int,int>{1,2};

Works with:

std::pair

std::tuple

structs



---

✅ if constexpr (Compile-time branching)

template<typename T>
void f(T x) {
    if constexpr (std::is_integral_v<T>) {
        // compiled only for integral types
    }
}

Impact

Replaces many SFINAE use cases

Cleaner template code



---

✅ Inline Variables

inline int x = 10;

Why it matters

Solves ODR issues for global variables

Important for header-only libraries



---

✅ Fold Expressions (Variadic Templates)

template<typename... Args>
auto sum(Args... args) {
    return (args + ...);
}

Types

Unary fold

Binary fold



---

✅ constexpr Improvements

if, switch, loops allowed more freely

More functions usable at compile-time



---

✅ Nested Namespace Definition

namespace A::B::C {
    // instead of namespace A { namespace B { namespace C {
}


---

✅ [[nodiscard]] Attribute

[[nodiscard]] int compute();

Prevents ignoring return values


---

✅ [[fallthrough]] Attribute

switch(x) {
    case 1:
        doSomething();
        [[fallthrough]];
    case 2:
        ...
}


---

✅ [[maybe_unused]]

Avoid unused warnings


---

✅ [[deprecated]] Improvements

[[deprecated("Use new_function instead")]]
void old_function();


---

✅ Guaranteed Copy Elision (BIG)

T x = T();  // no copy/move anymore

Impact

Performance boost

Changes object lifetime semantics



---

✅ Temporary Materialization Changes

Prvalues no longer create temporaries immediately

More optimization opportunities



---

✅ Evaluation Order Fixes

Function arguments evaluated left-to-right

operator<< chaining is now well-defined
