⚡ C++14 — Complete Feature List


---

🧠 1. Core Language Features


---

✅ Return Type Deduction for Functions

What changed

C++11:

auto f() -> int { return 10; }

C++14:

auto f() { return 10; }  // deduced automatically

Details

Works like auto variable deduction

Multiple return paths must match



---

✅ Generic Lambdas

What changed

Lambdas can now use auto parameters:

auto lambda = [](auto x, auto y) {
    return x + y;
};

Impact

Lambdas become function templates

Massive usability improvement



---

✅ Initialized Lambda Captures (Generalized Capture)

New capability

auto ptr = std::make_unique<int>(10);

auto lambda = [p = std::move(ptr)]() {
    return *p;
};

What it enables

Move-only captures

Custom expressions in capture

No need for workaround structs



---

✅ Relaxed constexpr (Huge usability improvement)

C++11 limitation

Only one return statement allowed


C++14 allows:

constexpr int factorial(int n) {
    int result = 1;
    for (int i = 2; i <= n; ++i)
        result *= i;
    return result;
}

New capabilities

Loops

Local variables

Multiple statements



---

✅ Variable Templates

New feature

template<typename T>
constexpr T pi = T(3.1415926535897932385);

Use case

Cleaner template constants

Used heavily in metaprogramming



---

✅ Binary Literals

int x = 0b101010;


---

✅ Digit Separators

int million = 1'000'000;


---

✅ decltype(auto)

What changed

decltype(auto) f() {
    return some_reference;
}

Why it matters

Preserves value category

Avoids accidental copies
